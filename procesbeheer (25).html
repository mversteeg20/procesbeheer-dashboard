🔑 Azure DevOps ● Lokaal

🔗 Koppelen aan Azure DevOps — stappenplan

Stap 1 — PAT aanmaken
  1. Ga naar fcseurope.visualstudio.com
  2. Klik rechtsboven op je profielfoto → Personal Access Tokens
  3. Klik New Token
  4. Naam: Procesbeheer Dashboard · Looptijd: 1 jaar
  5. Scope: Work Items → Read (alleen lezen is voldoende)
  6. Klik Create → kopieer de token (je ziet hem maar één keer)
⚠ Heb je eerder een PAT gedeeld in een chat of e-mail? Revoke die dan eerst via dezelfde pagina.
Stap 2 — PAT opslaan in dashboard
  1. Plak de PAT in het veld linksboven (Plak je Personal Access Token…)
  2. Klik Opslaan — de token wordt versleuteld opgeslagen in jouw browser
  3. Klik ↻ Statussen ophalen om alle ADO-statussen te laden
Stap 3 — Processen koppelen aan ADO
  1. Open een procesregel in de hiërarchietabel
  2. Klik op het 🔗 knopje rechts van de procesnaam
  3. Vul het work item ID in (bijv. 220030) of zoek op naam
  4. Klik Koppelen — de ADO-status verschijnt als badge naast de procesnaam
  5. Klik op een badge om het work item te openen in Azure DevOps
ℹ Het dashboard leest alleen uit ADO — wijzigingen aan work items doe je rechtstreeks in Azure DevOps.
